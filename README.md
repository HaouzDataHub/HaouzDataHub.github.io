# HaouzDataHub - Data Analyst Portfolio

## Overview

Welcome to **HaouzDataHub** - a professional portfolio website built to showcase data analysis expertise, case studies, and technical skills.

**Live Portfolio:** https://haouzdatahub.github.io

---

## Key Features

### Homepage
- Professional introduction and value proposition
- Interactive navigation to portfolio sections
- Contact links to LinkedIn, GitHub, and email
- Dark/Light mode toggle

### Skills Section
- Interactive Skills Dashboard with proficiency levels
- Admin authentication system (add/edit/delete skills)
- Dynamic skill cards with animations
- Organized by technology type

### Case Studies & Projects
- Detailed analysis projects (Cyclistic Bike Share Analysis)
- Links to GitHub repositories
- Interactive visualizations and dashboards

### About Section
- Professional background and career journey
- Key achievements and certifications

---

## Technology Stack

**Frontend:**
- HTML5 - Semantic markup
- CSS3 - styles.css, styles-modern.css, styles-skills.css
- JavaScript - script.js, skills-script.js, skills-data.js

**Deployment:**
- GitHub Pages
- GitHub Actions CI/CD
