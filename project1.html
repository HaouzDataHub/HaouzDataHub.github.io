Cyclistic Bike Share Case Study

Google Data Analytics Capstone Project by Hamza Haouzmani

About the Project

This case study analyzes bike usage data from Cyclistic to uncover insights that help convert casual riders into annual members. The project follows the full data analysis process: Ask, Prepare, Process, Analyze, Share, and Act.

Tools Used

  • R for data cleaning and analysis
  • Tableau for visualization
  • Excel / Google Sheets for exploration
  • GitHub for version control

Dashboard Preview

Key Insights

  • Casual riders mostly use bikes on weekends for leisure.
  • Members ride more frequently and use bikes for commuting.
  • Targeted marketing could convert casual riders to members through promotions and cost-saving plans.
