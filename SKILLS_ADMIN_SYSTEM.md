# Skills Page Admin System

## نظام إدارة صفحة المهارات (Skills Page Admin System)

### الوصف
نظام متكامل لإدارة بطاقات المهارات (Skill Cards) مع:
- تخزين مركزي في GitHub (skills-data.json)
- مصادقة إدارية قوية
- تحديث تلقائي عند التعديل/الإضافة/الحذف
- عرض موحد على جميع المتصفحات والأجهزة

### البنية

1. **skills-data.json** - ملف تخزين البيانات المركزي
   - يحتوي على جميع بطاقات المهارات
   - يُحدّث تلقائياً عند أي تعديل من قِبل الإدمن
   - متاح للقراءة العامة من المتصفحات

2. **skills-script.js** - سكريبت التطبيق
   - تحميل البيانات من skills-data.json
   - نظام مصادقة إداري
   - عمليات CRUD (Create, Read, Update, Delete)

### نظام المصادقة

#### كلمة المرور الإدمن
- تُخزن مشفرة في sessionStorage
- تُطلب عند محاولة: الإضافة، التعديل، الحذف
- صلاحيات الإدمن تنتهي عند إغلاق المتصفح (Session)

#### الصلاحيات
- ✅ **الإدمن فقط يستطيع:**
  - إضافة بطاقات جديدة
  - تعديل البطاقات الموجودة
  - حذف البطاقات
  - تحميل الصور

- ❌ **الزوار العام:**
  - عرض البطاقات فقط
  - نسخ الأكواد
  - البحث والتصفية
  - **لا يمكنهم:** الإضافة أو التعديل أو الحذف

### التعديل / الإضافة / الحذف

عند تنفيذ أي عملية تعديل:

1. التحقق من كلمة المرور الإدمن
2. التحديث في الواجهة (UI)
3. تحديث ملف `skills-data.json` على GitHub بشكل تلقائي
4. إعادة تحميل البيانات من الخادم للتأكد

### المزايا

✅ **المشاركة عبر المتصفحات**
- البيانات تُخزن في GitHub ومتاحة لجميع المتصفحات
- التحديثات فورية على جميع الأجهزة

✅ **الأمان**
- الزوار العام لا يستطيعون التعديل
- الإدمن فقط له الصلاحيات الكاملة
- كلمة المرور محمية

✅ **سهولة الإدارة**
- واجهة ويب بسيطة للإدارة
- تحديث تلقائي على GitHub
- بدون الحاجة لخادم باك-إند معقد

### خطوات الاستخدام

#### للإدمن:
1. افتح صفحة Skills
2. اضغط على "Add Skill" أو "Edit" أو "Delete"
3. أدخل كلمة المرور عند الطلب
4. أكمل العملية (الإضافة/التعديل/الحذف)
5. سيتم تحديث البيانات على GitHub تلقائياً

#### للزوار:
1. افتح صفحة Skills
2. استعرض البطاقات
3. انقر "View" لمشاهدة التفاصيل
4. استخدم "Copy" لنسخ الأكواد
5. **لا يمكنك الإضافة أو التعديل أو الحذف**
