### Update Personal Information

Edit `index.html` and modify:

```html
<!-- Name and Title -->
```

### Modify Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --color-primary: #1A1A1A;      /* Main text color */
    --color-secondary: #666666;     /* Secondary text */
    --color-background: #FAFAFA;    /* Page background */
    --color-accent: #0A0A0A;        /* Accent color */
}
```
